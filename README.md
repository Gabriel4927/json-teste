# Como procurar na pagina?
> Pressione "CTRL + F" e digite na caixa oque você quer
>
> Para uma pesquisa com mais precisão não inclua "-"
>
> Por exemplo, pesquise “doce” ou “flor” em vez de “flor-doce”.
